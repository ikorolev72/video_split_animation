Animation effect ASS developers guide

This doc describe how to do the text animation effect for video with ffmpeg and ASS subtitles. Docs for specified ASS effects available here. Mostly effective use for this the Aegisub tool for creating and modifying subtitles.
This format (ASS ) have 2 main sections: [V4+ Styles] and [Events] . In style sections we describe common style settings ( font, fontsize, color, bold, italian, etc), in event section we write the any text with (or without) additional properties, animation tranform, movement, etc .
Eg: 
[Script Info]
ScriptType: v4.00+
PlayResX: 480
PlayResY: 480
WrapStyle: 0
YCbCr Matrix: TV.601

[V4+ Styles]
Format: Name, Fontname, Fontsize, PrimaryColour, SecondaryColour, OutlineColour, BackColour, Bold, Italic, Underline, StrikeOut, ScaleX, ScaleY, Spacing, Angle, BorderStyle, Outline, Shadow, Alignment, MarginL, MarginR, MarginV, Encoding
Style: Default,Arial,40,&H00525252,&H0000FFFF,&H00FCFCFD,&H00000000,0,-1,0,0,100,100,0,0,1,2,0,8,0,0,300,1

[Events]
Format: Layer, Start, End, Style, Name, MarginL, MarginR, MarginV, Effect, Text
Dialogue: 0,0:00:00.00,0:00:05.00,textSkewShadowAnimation,,0,0,0,,Any text
Let's imagine we need to do the 'move' text effect from coordinates (100, 100) to coordinates (200, 200). Ok, let's prepare next ass file. I this case, we can add ASS tag move into dialog string, and text will move from initial point to end point. 
Dialogue: 0,0:00:00.00,0:00:05.00,textSkewShadowAnimation,,0,0,0,,{\move( 100,100, 200, 200)} Any text
Ok, what about fadeIn and fadeOut effect ? No problem: 
    Dialogue: 0,0:00:00.00,0:00:05.00,textSkewShadowAnimation,,0,0,0,,{\fad(500,500)} {\move( 100,100, 200, 200)} Any text
May be shadow or change alignment: 
    Dialogue: 0,0:00:00.00,0:00:05.00,textSkewShadowAnimation,,0,0,0,,{\shad10}{a6}{\fad(500,500)} {\move( 100,100, 200, 200)} Any text
See the full docs for specified ASS effects available here

Additional info:

Alignment

Alignment defined as digital keyboard ( digit from 1 to 9 ). There is the sample for 'text' with position in the center with different alignment

Colors

Color values are expressed in hexadecimal BGR format as &HBBGGRR& or ABGR (with alpha channel) as &HAABBGGRR&. Transparency (alpha) can be expressed as &HAA&. Note that in the alpha channel, 00 is opaque and FF is transparent

Special characters

Soft line break 
\n
Insert a forced line break, but only when in wrapping mode 2. (See the \q tag). Note that this is a lowercase n. In all other wrapping modes, this is replaced by a regular space. This is rarely (if ever) actually useful. If you're not sure whether you want this or \N, you probably want \N.

Hard line break 
\N
Insert a forced line break, regardless of wrapping mode. Note that this is an uppercase N.

Hard space 
\h
